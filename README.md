# Superstore-Analysis
📊 Power BI dashboard analyzing Super Store data using DAX functions to uncover sales, profit, and customer insights.


## 📌 Project Overview

This project showcases an interactive **Power BI Dashboard** built using the Super Store dataset to analyze business performance across sales, profit, customer segments, and regions. The goal of this project is to transform raw data into meaningful insights using **DAX functions** and data visualization techniques.

---

## 📊 Key Insights

* 💰 Total Sales: 1.92M
* 💵 Total Revenue: 1.98M
* 📉 Profit trends analyzed on a weekly basis
* 👥 Customer segment-wise profit comparison
* 🌍 Region-wise sales performance (East, West, Central, South)
* 🚚 Sales distribution based on shipping distance

---

## 🧠 DAX Functions Used

This project focuses on applying important **Date-Time and Text DAX functions**:

* **EOMONTH()** – Used for end-of-month calculations
* **WEEKNUM()** – Converts date into week number for trend analysis
* **FORMAT()** – Formats data for better readability
* **TRIM()** – Removes unwanted spaces from text
* **REPLACE()** – Replaces part of text with new value
* **SEARCH()** – Finds text position (case-insensitive)
* **FIND()** – Finds text position (case-sensitive)
* **SUBSTITUTE()** – Replaces specific text in a string
* **HOUR()** – Extracts hour from time
* **MINUTE()** – Extracts minutes from time
